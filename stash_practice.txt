let's learn stash in git.

git stash - to stash the changes
git stash pop - to apply the top stash
git stash list - to show stash list
git stash clear - to clear all stash list
git stash apply 'stash@{0}' - to apply specific index stash
git stash drop 'stash@{0}' - to drop speicfic stash
git stash save 'pdendi stash' - this will create a stash and saved with message.

These are few commands;
For documentation: https://git-scm.com/docs/git-stash
